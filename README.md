# Nutrition Tracker

A dual-role nutrition tracking platform connecting users with nutritionists. 
Users complete a guided onboarding profile (goals, activity level, training 
days, medical notes), and nutritionists build personalized meal plans with 
calorie and macro tracking.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features

**User Side**
- Multi-step onboarding form (personal info, weight/height, goals, activity level)
- Goal selection via interactive chips (lose weight, build muscle, etc.)
- Training days selector
- Medical notes and dietary preferences input
- Profile summary and submission confirmation

**Nutritionist Side**
- Client card view with full profile summary
- Day-by-day meal plan builder (breakfast, lunch, dinner, snacks)
- Per-meal calorie, protein, and carb tracking
- Live daily macro totals
- Save draft / send plan to client with notification
